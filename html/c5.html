Création d'un site web

Description globale

Pour un projet universitaire pendant ma 1ere année de BUT j'ai développé un site web pour les Jeux Olympiques de Paris 2024. En groupe de 4, nous avons développé le site en partant de la demande d'un client fictif.

Les grandes étapes

  1. Rédaction d'une synthèse structurée à partir de l’interview client
  2. Conception de l’arborescence du site (profondeur maximale de 3 niveaux) et définition des contenus de chaque page
  3. Élaboration d'une charte en lien avec l’univers des Jeux Olympiques
  4. Réalisation de maquettes pour les principaux formats (smartphone et desktop) sur Figma
  5. Création du site en HTML, CSS et JavaScript (optionnel), avec un focus sur la responsivité
