Réaliser un développement d'application

Snake

Description globale

Pour un projet universitaire pendant ma 1ere année de BUT j'ai développé un Snake jouable sur le terminal en langage C.

Les grandes étapes

  1. Implémentation initiale du jeu avec déplacement non contrôlé par l’utilisateur. Création du cahier de tests.
  2. Gestion des déplacements du serpent et de l'augmentation de la vitesse.
  3. Intégration d’obstacles et gestion des collisions.
  4. Finalisation du gameplay avec ajout des pommes.
  5. Génération de la documentation technique avec Doxygen.

Mes acquis

  • Maitrise langage C
  • Organisation
  • Gestion de la pression

Les outils utilisés

  • Langage C
  • Visual Studio Code
  • Doxygen
